# KoreaDoPiFamily（江原道親子行）

家人請開：**https://yeyejames.github.io/KoreaDoPiFamily/**

## 啟用 GitHub Pages（首次須操作一次）

網址要生效前，請在 GitHub 上開啟：

1. 進入本倉庫：<https://github.com/YeyeJames/KoreaDoPiFamily>
2. 點 **Settings**（設定）
3. 左側選 **Pages**
4. **Build and deployment** → **Source** 選 **Deploy from a branch**
5. **Branch** 選 **main**，資料夾選 **/ (root)** → **Save**
6. 約 1～3 分鐘後再開啟上方家人連結（若 404 請重新整理或稍等）

本站為靜態網頁（`index.html`），無資料庫；行程細節亦見倉庫內 PDF 與 `江原道五日遊_主要背景資料.md`。
